# Desktop Icon Assets

Generated desktop icon resources for the tray/frontend work.

- Source brand mark: `assets/branding/codex-remote-logo.svg`
- Generator: `python3 scripts/branding/generate_desktop_icons.py`
- Windows app/tray pack: `codex-remote-app.ico`
- macOS app icon pack: `codex-remote-app.icns` and `codex-remote-app.iconset/`
- macOS tray template source: `codex-remote-tray-template.svg`
- Generated macOS tray template rasters: `codex-remote-tray-template.png`, `codex-remote-tray-template@2x.png`
- Generated color tray rasters: `codex-remote-tray-color-*.png`
